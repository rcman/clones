# Binary Code Extractor

<BR>

The Zip file located at this link http://www.bitsavers.org/bits/UniversityOfHawaii/BCC-500/   Once extracted contains Binary files. This Javascript can extract those binary files and show you whats in them.
<BR>
Run the Binary_Extractor.html in your browser.
