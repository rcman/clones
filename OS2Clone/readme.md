Features:

Classic teal desktop background - iconic OS/2 look
Desktop icons - OS/2 Window, System Setup, Drive C:, and Information
Working terminal window with classic green-on-black text
OS/2 taskbar at the bottom with the red OS/2 logo
Start menu with classic options
Draggable windows - click and drag the title bar
Window controls - minimize, maximize, and close buttons

Terminal Commands:

help - Shows available commands
ver - Displays OS/2 version
dir - Lists directory contents
cls - Clears the screen

How to use:

Double-click desktop icons to open windows
Click the OS/2 button in the taskbar for the start menu
Drag windows by their title bars
Type commands in the terminal and press Enter

The interface captures that classic 90s IBM OS/2 Warp aesthetic with the gradient buttons, outset borders, and authentic color scheme!
