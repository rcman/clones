New Features:
1. User Management

useradd <username> - Creates new users (root only)

Automatically creates home directory in /home/<username>
Creates .bashrc file with welcome message
Creates empty .bash_history file
Prompts for password during creation


rmuser <username> - Removes users (root only)

Cannot remove root user
Updates /etc/passwd



2. Password System

Passwords stored as SHA-256 hashes in /etc/passwd
Format: username:password_hash:home_directory
passwd [username] - Change passwords

Users can change their own password
Root can change any user's password



3. User Sessions

Login screen at boot with username/password authentication
Each user has isolated command history (.bash_history)
history command shows last 100 commands
History persists across sessions

4. Per-User .bashrc

Executes automatically on login
Supports echo commands for welcome messages
Can be edited with vi or ed

5. Multi-UART Support

Two operating modes:

Single mode: One session on specified UART
Monitor mode: Monitors both UART0 and UART1 for connections


Uses threading to handle multiple simultaneous sessions
Each session is independent with separate user logins
